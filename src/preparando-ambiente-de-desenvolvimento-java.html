Preparando ambiente de desenvolvimento java

Comece a programar Java em 5 minutos!

Introdução

Neste tutorial você irá montar um ambiente completo para desenvolvimento de aplicações Java. Este tutorial é focado para os iniciantes neste ramo de TI ou programadores de outras linguagens.

Primeiramente você precisará instalar o Java na sua máquina e depois baixar alguma IDE (Integrated Development Environment). Uma IDE basicamente é um software que facilita diversas funções básicas que todo programador deve ter em mãos. Imagine a IDE como o Word para programadores.

Princípios basícos do Java

Primeiramente, você deve compreender as diferenças entre Java, JDK e JRE. JAVA é uma linguagem de programação orientada a objeto. Todas as aplicações feitas em java são primeiramente compiladas no que chamamos de bytecodes, que por sua vez são interpretados por uma máquina virtual.

Portanto para desenvolver qualquer aplicativo JAVA, você deve possuir, um compilador java (javac) e uma máquina virtual java (JVM). Ela está inclusa em qualquer pacote JRE (Java Runtime Enviroment) da Oracle. Para usuários comuns apenas a JRE já é suficiente, mas para desenvolvedores, um pacote mais elaborado, ou melhor, um pacote próprio para desenvolvimento JAVA é necessário. Este pacote é o JDK (Java Development Kit), que inclui, entre muitas outras coisas, o javac em seu pacote.

Começando

  • Acesse a página de Download do JDK www.oracle.com
  • Leia e depois marque a opção sobre os Termos de Licensa.
  • Selecione o sistema operacional do seu computador.
  • Clique no link com o ícone de download

Por exemplo, se seu sistema operacional é o Windows 8 64bits, clique no link jdk-8u60-windows-x64.exe

Para usuários Ubuntu, existe uma opção rápida pelo apt-get. Basta executar: sudo apt-get install default-jdk

Confirmando Instalação do JDK…

Após ter feito os passos descritos acima, vamos ter certeza que você fez a coisa certa. Abra um terminal de comando (famoso cmd do windows) ou algum terminal do sistema operacional Linux e digite: 

$ java -version

O resultado deve ser algo parecido com o abaixo:

$ java -version

java version "1.7.0_79"
Java(TM) SE Runtime Environment (build 1.7.0_79-b15)
Java HotSpot(TM) Client VM (build 24.79-b02, mixed mode)

Caso não apareça essas mensagens, repita os passos anteriores para certificar que a instalação foi realizada com sucesso. O comando descrito acima exibe na tela a versão do meu java, no caso 1.7.0.

Baixando Eclipse IDE

Vamos agora para a última parte que é o download do Eclipse. Para ser sincero essa é a parte mais fácil.

Apenas faça o download do Eclipse IDE for Java EE Developers, clicando neste link, descompacte o arquivo em alguma pasta e execute o eclipse.exe.

Após ter realizado todos os passos, parabéns, você está com todo o seu sistema pronto para desenvolver aplicativos Java.

Considerações Finais

  • A versão do Eclipse IDE for Java EE Developers é a mais completa de todas e totalmente gratuita. Já vem pré-instalado os melhores plugins para desenvolvimento Web e corporativo de aplicações Java.
  • Nos sistemas operacionais Linux que possuem gerenciamento de pacotes como o Ubuntu ou Fedora, é possível instalar o Eclipse diretamente pelo apt-get. Verifique a sintaxe correta de acordo com o S.O.
  • De uma olhada também em outras IDEs do mercado, como o Netbeans (possui versão gratuita) e o IntelliJ (possui versão paga e gratuita).
